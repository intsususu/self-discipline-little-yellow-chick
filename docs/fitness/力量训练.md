# Traditional Strength Training

精选大众、难度适中的传统力量训练动作，共 76 个（多为难度 2-3，含少量进阶；自重 / 自由器械 / 固定器械 / 壶铃）。**完整动作索引（全部 446 个）见 [all.md](videos/all.md)**；进阶 / 少见 / 小众动作见 [advanced.html](advanced.html)。

难度说明：1 入门/恢复，2 基础，3 中等，4 进阶，5 高阶。

> 注：其中 4 个动作暂无视频文件（表中显示 —）：杠铃卧推、杠铃深蹲、杠铃罗马尼亚硬拉、腿屈伸。

> 数据与 App 内「训练计划 · 力量训练」一致，由 scratchpad/export_index.py 从 TrainingPlanModels.swift 导出。
