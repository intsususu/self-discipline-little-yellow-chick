## <a id="action-001"></a>腹垫仰卧起坐

- 英文名：`Ab Mat Sit up`
- 难度：2/5
- 视频：[female](../../downloads/female/Ab%20Mat%20Sit-up.mp4) / [male](../../downloads/male/Ab%20Mat%20Sit-up.mp4)
- 动作描述：腹垫仰卧起坐是一个以自身体重或基础器械完成的腹部屈曲动作，通过卷起躯干或骨盆来刺激腹直肌，适合用于核心力量和腹部控制训练。
- 主要肌群：腹直肌
- 协同肌群：腹内外斜肌、腹横肌、髋屈肌群
- 动作要点：
  - 呼气卷起躯干，优先让肋骨向骨盆靠近。
  - 颈部放松，不用手猛拉头部或借甩臂起身。
  - 下放阶段保持控制，避免身体直接砸回地面。
  - 如果腰部代偿明显，缩短幅度或降低难度。

## <a id="action-004"></a>交替触踵

- 难度：2/5
- 视频：[female](../../downloads/female/Alternate-Heel-Touchers.mp4) / [male](../../downloads/male/Alternate-Heel-Touchers.mp4)
- 动作描述：交替触踵是一个侧向卷腹动作，通过上背轻微离地并左右触碰脚跟来训练腹斜肌和腹直肌控制。
- 主要肌群：腹内外斜肌
- 协同肌群：腹直肌、腹横肌、髋屈肌群
- 动作要点：
  - 上背轻微卷起后保持张力，避免反复完全放松。
  - 左右触踵时让肋骨向同侧骨盆靠近，不要只伸手够脚。
  - 颈部放松，视线保持自然，不用头部带动身体。
  - 动作节奏保持平稳，左右幅度尽量一致。

## <a id="action-005"></a>交替举腿

- 英文名：`Alternate Leg Raise`
- 难度：2/5
- 视频：[female](../../downloads/female/Alternate-Leg-Raise.mp4) / [male](../../downloads/male/Alternate-Leg-Raise.mp4)
- 动作描述：交替举腿是一个以自身体重或基础器械完成的举腿提膝类核心动作，通过骨盆后倾和下腹控制完成腿部移动，常用于强化下腹部和髋屈肌控制。
- 主要肌群：腹直肌、髂腰肌
- 协同肌群：腹横肌、腹内外斜肌、股直肌、前锯肌

  - 先做骨盆后倾，再抬腿或提膝，减少腰椎代偿。

  - 下放时控制离心，保持腹部持续发力。
  - 悬垂动作中先稳定肩胛，避免身体大幅晃动。
