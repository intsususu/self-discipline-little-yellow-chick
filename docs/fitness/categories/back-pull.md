## <a id="action-001"></a>交替背阔肌下拉

- 英文名：`Alternate Lateral Pulldown`
- 难度：3/5
- 视频：[female](../../downloads/female/Alternate-Lateral-Pulldown.mp4) / [male](../../downloads/male/Alternate-Lateral-Pulldown.mp4)
- 动作描述：交替背阔肌下拉是一个以自身体重或基础器械完成的垂直拉动作，重点训练背阔肌下拉和肩胛下沉能力，帮助提升引体、下拉和上肢拉力表现。
- 主要肌群：背阔肌
- 协同肌群：大圆肌、菱形肌、斜方肌中下束、肱二头肌、肱肌
- 动作要点：
  - 先下沉肩胛再拉动手肘，避免一开始就耸肩。
  - 把肘部向肋骨或髋部方向带动，感受背阔肌收缩。
  - 身体保持稳定，不用摆腿或甩身借力，除非动作本身要求摆动。
  - 还原时控制肩胛上旋，不要突然放松到底。

## <a id="action-002"></a>交替超人式

- 英文名：`Alternating Superman`
- 难度：4/5
- 视频：[female](../../downloads/female/Alternating-Superman.mp4) / [male](../../downloads/male/Alternating-Superman.mp4)
- 动作描述：交替超人式是一个以自身体重完成的后链稳定动作，通过交替抬起对侧手脚训练下背、臀部和肩胛控制。
- 主要肌群：竖脊肌
- 协同肌群：臀大肌、腘绳肌、多裂肌、肩胛稳定肌群
- 动作要点：
  - 抬起手脚时保持骨盆稳定，避免身体向一侧翻转。
  - 只抬到下背可控的高度，不用猛甩手脚。
  - 顶点短暂停留，感受臀部和背部后侧发力。
  - 下放时保持控制，让呼吸和动作节奏一致。
