## <a id="action-001"></a>空中蹬车

- 英文名：`Air Bike`
- 难度：2/5
- 视频：[female](../../downloads/female/Air-Bike.mp4) / [male](../../downloads/male/Air-Bike.mp4)
- 动作描述：空中蹬车是一个有氧、敏捷或跳跃类动作，通过连续节奏、快速换步或弹性发力提升心肺能力、协调性和下肢反应速度。
- 主要肌群：髂腰肌、腹直肌
- 协同肌群：腹内外斜肌、股四头肌、腘绳肌
- 动作要点：
  - 保持稳定节奏，动作幅度以能连续控制为准。
  - 落地轻盈，膝盖和脚尖方向一致，避免膝内扣。
  - 核心收紧，躯干不要无序晃动。
  - 根据心肺能力调整速度和组间休息，先保证动作质量。
