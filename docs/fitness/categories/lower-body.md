## <a id="action-001"></a>徒手深蹲

- 英文名：`Air Squat`
- 难度：3/5
- 视频：[female](../../downloads/female/Air-Squat.mp4) / [male](../../downloads/male/Air-Squat.mp4)
- 动作描述：徒手深蹲是一个以自身体重或基础器械完成的下肢复合动作，主要围绕髋、膝、踝协同发力，提升腿部力量、臀腿控制和单/双腿稳定性。
- 主要肌群：股四头肌、臀大肌
- 协同肌群：腘绳肌、内收肌群、臀中肌、核心稳定肌群
- 动作要点：
  - 脚掌三点着地，膝盖方向与脚尖保持一致。
  - 下蹲时同时屈髋屈膝，保持躯干稳定和核心收紧。
  - 底部不要塌腰或膝内扣，按可控深度完成。
  - 起身时脚跟和前脚掌均匀发力，髋膝同步伸展。

## <a id="action-003"></a>弹力带提踵

- 英文名：`Band Calf Raise`
- 难度：2/5
- 视频：[female](../../downloads/female/Band-Calf-Raise.mp4) / [male](../../downloads/male/Band-Calf-Raise.mp4)
- 动作描述：弹力带提踵是一个使用弹力带完成的小腿训练动作，通过踝关节跖屈强化腓肠肌、比目鱼肌和足踝稳定。
- 主要肌群：腓肠肌、比目鱼肌
- 协同肌群：胫骨后肌、腓骨肌群、足底稳定肌群
- 动作要点：
  - 通过踝关节完成动作，膝髋保持稳定。
  - 顶点充分跖屈并短暂停留，感受小腿收缩。
  - 下放时控制速度，允许小腿被温和拉长。
  - 保持脚踝正直，不要向内或向外塌陷。
  - 使用弹力带时先确认握距、站位和重量适合当前动作。

## <a id="action-004"></a>弹力带深蹲划船

- 英文名：`Band Squat Row`
- 难度：3/5
- 视频：[female](../../downloads/female/Band-Squat-Row.mp4) / [male](../../downloads/male/Band-Squat-Row.mp4)
- 动作描述：弹力带深蹲划船是一个使用弹力带完成的复合训练动作，把下肢深蹲和上肢划船结合在一起，训练髋膝伸展、背部拉力和核心稳定。
- 主要肌群：股四头肌、臀大肌、背阔肌、菱形肌
- 协同肌群：腘绳肌、臀中肌、斜方肌中下束、核心稳定肌群、肱二头肌
- 动作要点：
  - 下蹲时脚掌踩稳，膝盖方向与脚尖一致。
  - 起身或稳定站姿后完成划船，先收肩胛再带动手肘后拉。
  - 躯干保持中立，不要用圆背或后仰来代偿拉力。
  - 深蹲和划船节奏保持清晰，避免弹力带把身体拉失衡。
  - 使用弹力带时先确认握距、站位和阻力适合当前动作。

- 主要肌群：臀大肌、腘绳肌

  - 全程不要用腰部猛拉，先降低重量再追求幅度。
