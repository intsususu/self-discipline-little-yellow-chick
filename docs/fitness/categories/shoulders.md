## <a id="action-001"></a>绳索前平举

- 英文名：`Cable Front Raise`
- 难度：2/5
- 视频：[female](../../downloads/female/Cable%20Front%20Raise.mp4) / [male](../../downloads/male/Cable%20Front%20Raise.mp4)
- 动作描述：绳索前平举是一个使用绳索完成的肩部训练动作，强调肩胛稳定下的上举、推举或旋转控制，用于提升肩部力量、活动度和稳定性。
- 主要肌群：三角肌前束
- 协同肌群：胸大肌上束、斜方肌上束、核心稳定肌群
- 动作要点：
  - 肋骨下沉、核心收紧，避免通过后仰代偿完成动作。
  - 肩胛自然上旋，动作中避免耸肩卡住颈部。
  - 手腕保持中立，重量路径稳定可控。
  - 选择不会引发肩前侧夹挤感的活动范围。
  - 使用绳索时先确认握距、站位和重量适合当前动作。

## <a id="action-002"></a>绳索侧平举

- 英文名：`Cable Lateral Raise`
- 难度：2/5
- 视频：[female](../../downloads/female/Cable%20Lateral%20Raise.mp4) / [male](../../downloads/male/Cable%20Lateral%20Raise.mp4)
- 动作描述：绳索侧平举是一个使用绳索完成的肩部孤立动作，通过肩关节外展把手臂向两侧抬起，重点孤立刺激三角肌中束。
- 主要肌群：三角肌中束
- 协同肌群：三角肌前束、斜方肌上束、冈上肌
- 动作要点：
  - 手肘保持微屈并固定，靠肩部外展把手柄向两侧抬到与肩同高。
  - 肩胛下沉稳定，避免耸肩用斜方肌上束代偿。
  - 不靠后仰或甩动借力，顶点稍停后控制离心慢放。
  - 手腕保持中立，重量路径稳定可控。
  - 使用绳索时先确认握距、站位和重量适合当前动作。

## <a id="action-004"></a>绳索直立划船

- 英文名：`Cable Upright Row`
- 难度：3/5
- 视频：[female](../../downloads/female/Cable%20Upright%20Row.mp4) / [male](../../downloads/male/Cable%20Upright%20Row.mp4)
- 动作描述：绳索直立划船是一个使用绳索完成的肩部拉力动作，沿身体把肘部向上提拉，主要刺激三角肌中束和斜方肌上束。
- 主要肌群：三角肌中束、斜方肌上束
- 协同肌群：三角肌后束、肱二头肌、前臂屈肌群
- 动作要点：
  - 肘部沿身体上提并高于手腕，把手柄拉到上胸/锁骨高度。
  - 肩胛稳定，避免靠耸肩或后仰借力。
  - 不追求过高的拉起高度，感到肩前夹挤就降低幅度。
  - 下放阶段控制离心，保持重量贴近身体。
  - 使用绳索时先确认握距、站位和重量适合当前动作。
