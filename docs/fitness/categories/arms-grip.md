## <a id="action-001"></a>弹力带二头弯举

- 英文名：`Band Biceps Curl`
- 难度：2/5
- 视频：[female](../../downloads/female/Band-Biceps-Curl.mp4) / [male](../../downloads/male/Band-Biceps-Curl.mp4)
- 动作描述：弹力带二头弯举是一个使用弹力带完成的手臂孤立动作，围绕肘关节屈伸或腕部控制进行训练，适合补强上肢局部力量。
- 主要肌群：肱二头肌
- 协同肌群：肱肌、肱桡肌、前臂屈肌群
- 动作要点：
  - 上臂尽量固定，主要让肘关节完成屈伸。
  - 向上弯举时呼气，顶点短暂停留感受二头肌收缩。
  - 下放阶段放慢速度，避免靠摆动或耸肩借力。
  - 手腕保持中立或略微旋后，不要过度折腕。
  - 使用弹力带时先确认握距、站位和重量适合当前动作。
