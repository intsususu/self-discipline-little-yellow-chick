## <a id="action-001"></a>弓箭手俯卧撑

- 英文名：`Archer Push up`
- 难度：4/5
- 视频：[female](../../downloads/female/Archer%20Push-up.mp4) / [male](../../downloads/male/Archer%20Push-up.mp4)
- 动作描述：弓箭手俯卧撑是一个以自身体重或基础器械完成的推类动作，利用上肢推离身体或负重，发展胸部、肩前束和肱三头肌的协同发力。
- 主要肌群：胸大肌
- 协同肌群：三角肌前束、肱三头肌、前锯肌、肩袖肌群
- 动作要点：
  - 肩胛保持稳定，推起时不要耸肩顶颈。
  - 肘部路线与动作目标匹配，避免过度外展造成肩前侧压力。
  - 下降阶段控制速度，底部保持胸肩张力。
  - 推起时呼气发力，手腕保持中立并与前臂对齐。

## <a id="action-002"></a>手臂交叉摆动

- 难度：2/5
- 视频：[female](../../downloads/female/Arm-Crossover.mp4) / [male](../../downloads/male/Arm-Crossover.mp4)
- 动作描述：手臂交叉摆动是一个以自身体重或基础器械完成的胸部夹合动作，主要训练胸大肌在水平内收中的发力，适合作为胸部孤立或补充训练。
- 主要肌群：胸大肌
- 协同肌群：三角肌前束、肱三头肌、前锯肌、肩袖肌群
- 动作要点：
  - 手肘保持微屈，动作弧线稳定，不要变成推举。
  - 在胸部被拉长的位置保持控制，避免肩前侧被动拉扯。
  - 夹合时想象上臂向身体中线靠近，而不是只移动手掌。
  - 重量选择以胸部发力清晰为准，避免借力摆动。
