## <a id="action-001"></a>平衡板训练

- 英文名：`Balance Board`
- 难度：2/5
- 视频：[female](../../downloads/female/Balance-Board.mp4) / [male](../../downloads/male/Balance-Board.mp4)
- 动作描述：平衡板训练是一个全身功能性动作，通常结合髋腿发力、核心传导和上肢输出，训练爆发力、协调性或搬运能力。
- 主要肌群：臀大肌、股四头肌、三角肌
- 协同肌群：腘绳肌、背阔肌、斜方肌、核心稳定肌群、前臂握力肌群
- 动作要点：
  - 先由髋腿发力，再通过核心把力量传导到上肢。
  - 保持脊柱中立和躯干张力，避免用腰部硬扭硬拉。
  - 动作节奏要连贯，但每一次发力都要落在稳定姿态上。
  - 负重或速度提升前，先确认落地、接杠或接球位置可控。
