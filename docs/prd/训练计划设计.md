# 训练计划 PRD（交互设计）

> 目标：在「我的 → 数据与偏好 → 小工具」下提供一个**按部位分页的健身动作图解**工具。覆盖练背 / 练胸 / 练腿 / 练肩四个部位，每个部位一页，呈现该部位的主要训练器械图 + 对应肌肉发力图，并配文字说明目标肌群与动作要点。
> 状态：**交互设计（待落地）**。当前「小工具」分组中「训练计划」为占位入口（`placeholderToast`），本文档定义其完整交互、内容与素材需求。
> 入口：[ProfileView.swift](../HealthApp/Features/Profile/ProfileView.swift) `toolsSettings` → `训练计划` 行。
> 关联：同属「小工具」分组的还有 [食品热量表设计.md](食品热量表设计.md)。

## 1. 范围与定位

- **图解参考工具**，非计划编排/打卡/计时器。本期不做训练记录、组数计数、休息计时。
- 内容为**四个部位各一页**：练背 / 练胸 / 练腿 / 练肩；每页是该部位的「主要动作图鉴」。
- 全中文界面，风格与 App 现有卡片/分组一致。
- 底部固定免责声明：「训练动作仅供参考，请量力而行，必要时在专业指导下进行」。

## 2. 页面结构

### 2.1 顶层：训练计划页（单页 + 四部位切换）

从「我的」push 进入 `TrainingPlanView`：

1. **导航头**：左侧返回箭头（`<`，返回「我的」），居中标题「训练计划」。
2. **部位切换 Tab（横向胶囊，单选）**：`练背 / 练胸 / 练腿 / 练肩`，默认选中**练背**。选中态为蓝色实心胶囊（复用食品热量表的一级分类胶囊样式）。
3. **部位内容区**（随 Tab 切换，见 §2.2），可纵向滚动。
4. **底部免责声明**。

> 交互口径：四个部位为「一个工具内的 4 个分页」，通过顶部 Tab 切换（也支持左右滑动切页，见 §5）。这样满足「每一项一页」，又不必在「我的」里铺 4 个入口。

### 2.2 部位页内容（以「练背」为例）

自上而下：

1. **部位概览卡**
   - 部位名（如「练背」）+ 目标肌群标签组（如 `背阔肌` `斜方肌` `菱形肌` `竖脊肌`）。
   - **整体肌肉发力图**：一张人体（背面）轮廓图，高亮该部位主要发力肌群（大图，约 4:3 或 16:9）。
   - 一句话部位简介（如「背部是上肢拉的主力，练好背能改善体态、强化引体与硬拉表现」）。

2. **动作卡列表**（每个动作一张卡片）：
   - **器械/动作图**：健身房该动作的器械或动作示意图。
   - **肌肉发力图**：该动作主要发力肌群的高亮小图（与器械图并排或上下排布）。
   - **动作名**（如「高位下拉」）+ 英文名（可选，如 Lat Pulldown）。
   - **目标肌群**：主练 + 协同（如「主：背阔肌；协同：肱二头肌、后束」）。
   - **要点**：2–4 条 bullet（发力、轨迹、常见错误）。
   - **建议组数 × 次数**（可选字段，如「3–4 组 × 10–12 次」）。

3. 列表底部留白 + 全局免责声明。

> 可选交互：点击任一图片**全屏放大**查看（pinch/双击缩放）。本期可先不做，标记为后续增强。

## 3. 四个部位的内容清单

> 以下为内置文案与「需要的图」清单。动作以健身房常见器械/自由重量为主，每部位 4–5 个主力动作。数值/要点为大众健身常识，落地前可由产品再校。

### 3.1 练背 🦾（人体背面发力图）

**目标肌群**：背阔肌（主）、斜方肌、菱形肌、大圆肌、竖脊肌；肱二头肌为协同。

| 动作 | 英文 | 主练肌群 | 要点 | 建议 |
|---|---|---|---|---|
| 高位下拉 | Lat Pulldown | 背阔肌 | 沉肩后缩肩胛 · 肘部下沉带动 · 拉至上胸 · 顶峰收缩不耸肩 | 3–4×10–12 |
| 坐姿划船 | Seated Cable Row | 中背 / 菱形肌 / 背阔肌 | 挺胸收腹 · 肩胛后夹 · 肘贴身后拉 · 不靠腰甩动 | 3–4×10–12 |
| 杠铃俯身划船 | Barbell Row | 背阔肌 / 斜方肌中下 | 屈髋上身约 45° · 核心收紧 · 拉向下腹 · 背部全程挺直 | 3–4×8–10 |
| 引体向上 | Pull-up | 背阔肌 | 全握距 · 肩胛先下沉 · 拉到下巴过杠 · 控制离心下放 | 3–4×力竭 |
| 硬拉（可选） | Deadlift | 竖脊肌 / 后链 | 杠贴小腿 · 髋膝同步 · 背部中立 · 杠铃贴身上提 | 3–4×5–8 |

### 3.2 练胸 💪（人体正面发力图）

**目标肌群**：胸大肌（上 / 中 / 下束）、胸小肌、三角肌前束；肱三头肌为协同。

| 动作 | 英文 | 主练肌群 | 要点 | 建议 |
|---|---|---|---|---|
| 杠铃卧推 | Barbell Bench Press | 胸大肌中部 | 肩胛后缩下沉 · 杠落于乳线 · 肘约 45° · 脚踩稳发力 | 3–4×8–10 |
| 上斜哑铃卧推 | Incline DB Press | 胸大肌上束 | 椅背 30–45° · 下放至胸侧 · 顶端微内夹 · 不耸肩 | 3–4×10–12 |
| 坐姿推胸器 | Chest Press Machine | 胸大肌 | 背贴靠垫 · 握把与胸齐 · 推到near直臂 · 控制回放 | 3×12 |
| 蝴蝶机夹胸 | Pec Deck Fly | 胸大肌内侧 | 微屈肘固定 · 走弧线内夹 · 顶峰挤压 · 离心慢放 | 3×12–15 |
| 双杠臂屈伸（可选） | Chest Dips | 胸大肌下束 | 身体前倾 · 下放至肩略低于肘 · 不耸肩 · 控制幅度 | 3×力竭 |

### 3.3 练腿 🦵（人体正/背面发力图）

**目标肌群**：股四头肌、臀大肌、腘绳肌、内收肌、小腿（腓肠肌 / 比目鱼肌）。

| 动作 | 英文 | 主练肌群 | 要点 | 建议 |
|---|---|---|---|---|
| 杠铃深蹲 | Barbell Squat | 股四头 / 臀大肌 | 核心收紧 · 膝与脚尖同向 · 蹲至大腿平行 · 脚跟发力站起 | 3–4×8–10 |
| 腿举 | Leg Press | 股四头 / 臀 | 脚放踏板中部 · 膝不锁死 · 下放至约 90° · 不塌腰 | 3–4×10–12 |
| 腿屈伸 | Leg Extension | 股四头（孤立） | 调好转轴对齐膝 · 顶端伸直挤压 · 慢放 · 不甩腿 | 3×12–15 |
| 腿弯举 | Leg Curl | 腘绳肌 | 髋贴垫 · 收缩到底 · 离心控制 · 避免拱腰借力 | 3×12–15 |
| 罗马尼亚硬拉 | RDL | 腘绳肌 / 臀 | 屈髋后送 · 杠贴腿下滑 · 背中立 · 感受腘绳拉伸 | 3×10 |
| 站姿提踵（可选） | Calf Raise | 小腿 | 全幅度起落 · 顶端停顿 · 慢速离心 | 3×15–20 |

### 3.4 练肩 🤸（人体正/背面发力图）

**目标肌群**：三角肌前束 / 中束 / 后束；斜方肌、外旋肌群为协同。

| 动作 | 英文 | 主练肌群 | 要点 | 建议 |
|---|---|---|---|---|
| 哑铃肩上推举 | Overhead Press | 前束 / 中束 | 核心收紧 · 推过头顶 · 不过度后仰 · 控制下放 | 3–4×8–12 |
| 哑铃侧平举 | Lateral Raise | 中束 | 小重量 · 肘略高于腕 · 起到肩高 · 慢放不借力 | 3–4×12–15 |
| 俯身飞鸟 / 反向蝴蝶机 | Rear Delt Fly | 后束 | 微屈肘 · 肩胛勿过度后缩 · 走弧线 · 孤立后束 | 3×15 |
| 坐姿推肩机 | Shoulder Press Machine | 三角肌整体 | 背贴靠垫 · 握把与肩齐 · 推到near直臂 · 不耸肩 | 3×10–12 |
| 面拉（可选） | Face Pull | 后束 / 外旋肌 | 绳索高位 · 拉向面部并外旋 · 注重收缩 | 3×15 |

## 4. 素材需求（图片）

每个部位需要：

- **1 张部位整体肌肉发力图**（人体正面或背面轮廓，高亮该部位主肌群）。
- **每个动作 2 张图**：① 器械/动作示意图；② 该动作主肌群高亮的肌肉发力图。

**数量估算**：4 个部位 × (1 概览图 + 约 5 动作 × 2 图) ≈ **44 张图**。

**命名建议**（放入 `Assets.xcassets`，按部位前缀）：

```
back_overview_muscle           // 练背 概览发力图
back_lat_pulldown_equip        // 高位下拉 器械图
back_lat_pulldown_muscle       // 高位下拉 发力图
chest_bench_press_equip / _muscle
legs_squat_equip / _muscle
shoulder_overhead_press_equip / _muscle
...
```

**素材来源（待定，需注意版权）**：自制插画 / 购买图库授权 / 采用可商用的开源解剖图 / AI 生成统一风格图。**务必确认可商用授权**后再入库。

## 5. 交互细节

- **部位切换**：顶部 Tab 单选；支持内容区左右滑动（`TabView .page` 或手势）切换部位，与 Tab 同步高亮。
- **滚动**：每个部位页内容纵向滚动；切换部位时回到该页顶部。
- **图片放大**（可选 / 后续）：点击图片全屏查看，支持缩放。
- 无搜索、无排序（内容为固定图鉴）。

## 6. 数据模型（建议）

```
TrainingPart {
  id, name,                 // 「练背」
  targetMuscles: [String],  // 目标肌群标签
  intro: String,            // 一句话简介
  overviewImage: String,    // 概览发力图资源名
  exercises: [Exercise]
}
Exercise {
  id, name, nameEn?,
  primaryMuscles: String,   // 主练
  synergistMuscles: String?,// 协同
  equipImage: String,       // 器械/动作图资源名
  muscleImage: String,      // 发力图资源名
  points: [String],         // 要点 bullet
  setsReps: String?         // 「3–4 组 × 10–12 次」
}
```

内置静态数据：随包 Swift 字面量或 `training_plan.json`。
